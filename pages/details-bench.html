Which benchmark didi we use?

FrontierMath

350 expert-written problems in advanced mathematics, requiring multiple hours or even days to solve.

More details →

GPQA Diamond

A challenging multiple-choice question set in biology, chemistry, and physics, authored by PhD-level experts.

More details →

OTIS Mock AIME 2024-2025

45 competition-style math problems from OTIS, harder than MATH Level 5 but easier than FrontierMath.

More details →

SimpleQA Verified

1,000 factoid questions about politics, science and technology, art, sports, geography, music, and more.

More details →

Chess Puzzles

100 novel puzzles, generated programmatically with a chess engine. Each puzzle has a single best next move.

More details →

METR Time Horizons

Durations of the longest task that models can complete correctly more often than not, across a set of software engineering and related tasks.

More details →

SWE-bench Verified

500 GitHub issues from real-world Python repos, testing whether models can implement valid code fixes. Note: we have paused running our own evaluations on SWE-bench Verified while we upgrade our scaffold. Until then, for ECI, we are using data from the SWE-bench team's runs using mini-SWE-agent.
